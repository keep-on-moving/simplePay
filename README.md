# simplePay
基于tp5.1的简单支付（包含微信和支付宝）

#composer 安装

`composer require simple-pay/simple-pay` 

#使用方法
- 引入微信包

```
use Payment\Wechat;

$wechat = new Wechat('自己的APPid', '商户号', 'key', 'appsecret');

```
